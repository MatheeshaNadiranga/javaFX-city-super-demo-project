What is this Project ? City Super A Desktop Point of Sale (POS) system built with JavaFX and MySQL.

This is a practice project created to learn and implement Layered Architecture. It features a full billing system, customer management, and inventory tracking, demonstrating how to decouple the User Interface from the Database using DAO and Service layers.

Technologies: Java, JavaFX, JDBC, MySQL.

how to run this ? Install JDK and Run.
